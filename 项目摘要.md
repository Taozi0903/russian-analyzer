# 俄语翻译与语法分析器 - 项目摘要

## 项目概述
一个基于 Streamlit 的俄语学习辅助工具，支持俄中互译、俄语句式分析和逐词语法拆解。

## 核心架构

### 技术栈
| 层级 | 技术 |
|------|------|
| 前端框架 | Streamlit 1.59 |
| 本地词法分析 | pymorphy3（俄语形态分析） |
| 本地翻译 | Ollama + qwen2.5:1.5b（可选） |
| 云端翻译 | DeepSeek V4 Pro/Flash API |
| 图片处理 | Pillow |

### 分析引擎设计（三层架构）

```
输入句子
├── 第一层：翻译
│   ├── Ollama 本地模型（免费，需安装）
│   └── DeepSeek API（按 token 计费）
├── 第二层：句式分析
│   ├── 自动拆句（. ! ? 断句）
│   ├── 主句/从句识别（连接词 что/когда/если/который 等）
│   ├── 每句输出：主语 + 谓语 + 宾语/状语结构
│   └── 整段展示，不拆词
└── 第三层：逐词分析
    ├── pymorphy3 规则引擎（60,000+ 俄语词库）
    ├── 启发式后缀推断（4-5字符长后缀匹配）
    ├── LLM 补全未知词（仅必要时调用）
    ├── 词性→语法属性自动展开
    │   ├── 名词：性/数/格/动物性/变格法
    │   ├── 动词：体/及物性/接格关系/反身性/人称/时态/式
    │   ├── 形容词：性/数/格/长尾短尾
    │   ├── 前置词：支配格 + 用法说明
    │   ├── 连接词/语气词：类型 + 用法说明
    │   └── 副词：类别 + 级别
    ├── 中文翻译（内置120+高频词词典）
    ├── Wiktionary 外链
    ├── 标点与单词分离处理（防止分词失败）
    └── 分页显示（10词/页，可翻页）
```

## 核心功能清单

1. **俄→中翻译**：输入俄语句子，输出中文译文
2. **中→俄翻译**：输入中文句子，LLM 翻译后自动分析俄语结果
3. **句式分析**：自动识别主句/从句，整段截取展示结构
4. **逐词分析**：每个词的性数格、体态时、用法说明、翻译
5. **功能词用法**：内置 80+ 前置词/连接词/语气词用法说明
6. **常见副词释义**：40+ 时间/地点/方式/程度副词中文释义
7. **LLM 补全未知词**：本地分析未识别时自动调用 LLM 补全
8. **翻译评分**（可选）：5维度（语义/通顺/语体/文化/修辞）
9. **历史记录**：保存最近20条分析，FIFO队列，可回看
10. **自定义背景**：支持导入本地图片作为网页背景
11. **界面美化**：俄三色旗装饰线、渐变按钮、深色主题

## 三种运行模式

| 模式 | 语法分析 | 翻译 | 费用 |
|------|---------|------|------|
| 本地 + Ollama | pymorphy3 本地 | Ollama qwen2.5:1.5b | 零 |
| 本地 + DeepSeek | pymorphy3 本地 | DeepSeek API | 极少 |
| 纯 DeepSeek | DeepSeek 一次性分析 | DeepSeek | 正常 |

## 关键设计决策

1. **本地分析优先**：pymorphy3 提供确定性词法分析，零幻觉
2. **分层分析**：翻译→句式→逐词，层层递进
3. **启发式后缀匹配**：覆盖 pymorphy3 词库外的词（专名、生僻词）
4. **标点分离处理**：解决句中标点导致分词失败的核心问题
5. **LLM 仅作翻译**：本地分析引擎负责语法，LLM 只负责翻译
6. **多种部署方式**：纯本地/混合/纯云端，适应不同场景
